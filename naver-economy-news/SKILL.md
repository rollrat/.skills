# 네이버 경제 뉴스 요약 스킬

## 트리거 조건

사용자가 다음과 같은 요청을 할 때 이 스킬을 자동으로 활성화합니다:
- "경제 뉴스 요약", "경제 뉴스 알려줘"
- "오늘 시장 동향", "시장 분석"
- "코스피 분석", "코스피 동향", "코스닥 동향"
- "나스닥 동향", "미국 증시"
- "오늘 경제 뉴스", "경제 소식"
- "증권 뉴스", "금융 뉴스"
- "부동산 뉴스", "부동산 동향"
- "네이버 경제 뉴스"

## 실행 절차

1. Python 크롤러를 실행하여 네이버 경제 뉴스를 수집합니다:

```bash
python3 ~/.claude/skills/naver-economy-news/scripts/crawl_naver_economy.py
```

2. 생성된 파일(`/tmp/naver_economy_news_YYYYMMDD.md`, 오늘 날짜 기준)을 읽습니다.

3. 뉴스 데이터를 분석하여 다음 항목으로 요약합니다:
   - **전반적 시장 동향**: 국내외 경제 흐름 개요
   - **나스닥/미국 시장 동향**: 미국 증시, 주요 지표
   - **코스피/코스닥 동향**: 국내 증시, 수급 동향
   - **섹터별 주요 뉴스**: 금융, 증권, 산업/재계, 부동산, 글로벌경제, 생활경제, 중기/벤처
   - **주요 이슈 및 투자 시사점**: 핵심 이슈와 투자 관점 분석

4. 사용자의 질문이 특정 섹터나 주제에 대한 것이면 해당 부분을 더 상세하게 분석합니다.

## 주의사항
- 모든 요약은 한국어로 작성합니다.
- 뉴스가 없는 섹터는 생략합니다.
- 크롤링 오류 발생 시 사용자에게 명확히 안내합니다.
